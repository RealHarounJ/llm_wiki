# 📚 Source — Fundamentals of Database Systems

A comprehensive course covering the foundational concepts of databases, data modeling, and SQL.

## Curriculum Overview
1. **Chapter 1**: [[DBMS_Basics]] — Data, Mini-world, Metadata, and Abstraction.
2. **Chapter 2**: [[Data_Models_and_Architecture]] — 3-Schema Architecture, DDL/DML, and N-Tier Systems.
3. **Chapter 3**: [[ER_Modeling]] — Entities, Attributes, and the ER Diagram.
4. **Chapter 4**: [[Relationships_and_Constraints]] — Cardinality, Participation, and Weak Entities.
5. **Chapter 5**: [[Enhanced_ER_Model]] — Specialization and Generalization.
6. **Chapter 6**: [[SQL_Language_Basics]] — Introduction to Structured Query Language.

## Summary
The course emphasizes the **Database Approach**, which provides self-describing systems, program-data independence, and data abstraction. It moves from conceptual design (ER) to logical design (Database Schema) and physical design (Internal Storage).
