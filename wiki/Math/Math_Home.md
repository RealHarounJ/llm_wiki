# 📐 Mathematics — Exam Preparation Hub

## 🎯 Exam Structure

| Test | Duration | Threshold | Scoring |
|:---|:---|:---|:---|
| **1. Multiple Choice** | 30 min | ≥ 50% to access Test 2 | ✅ Correct = +points / ❌ Wrong = −points / ⬜ Blank = 0 |
| **2. Written Exercises** | 1 hour | Must pass to pass the exam | Open-ended problems |

> [!CAUTION]
> ❌ No calculators, phones, smartwatches, or any material allowed.
> ❌ If you **fail the written test**, you must **retake BOTH tests** in a future session.
> ❌ Answer sheets without **Student ID + Exam Key** receive NO marks.

> [!TIP]
> **Multiple choice strategy:** If unsure → **leave blank** (0 points). Only answer when reasonably confident. Wrong = negative score.

## 🗂️ Study Strategy

### Multiple Choice Tactics
- [ ] Master all **definitions** (vocab-heavy questions)
- [ ] Know all **standard formulas** by heart (no calculator)
- [ ] Practice **elimination**: rule out 2 wrong answers first
- [ ] **Never guess randomly** — negative scoring punishes it

### Written Test Tactics
- [ ] Always **state the method** before computing (partial credit)
- [ ] Check **domain restrictions** before solving
- [ ] Leave **30% of time** for review
- [ ] Write **Student ID + name** on EVERY sheet immediately
