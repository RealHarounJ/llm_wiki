# IO 08: Basic Econometrics (Lesson 11)

## Types of Data
- **Cross-Sectional**: Multiple units (firms, countries) at a *single point in time* (e.g., Sales of different firms in 1995).
- **Time-Series**: A single unit tracked over *multiple time periods* (e.g., Sales of Ferrari from 1995 to 2020).

## Linear Regression Model
Formula: $Y = \alpha + \beta X + \mu$
- $Y$: Dependent variable (what we want to explain).
- $X$: Independent/explanatory variable.
- $\alpha$: Intercept.
- $\beta$: Regression coefficient. If X increases by 1 unit, Y changes by $\beta$ units.
- $\mu$: Error term (unobserved factors).

## Statistical Inference
- **Statistical Significance**: Tests if $\beta$ is different from zero. Evaluated using t-statistics and standard errors (SE). Small SE implies high reliability. Common significance thresholds are 1%, 5%, 10% (denoted by asterisks ***).
- **R-squared ($R^2$)**: Coefficient of determination. Measures Goodness of Fit. It represents the proportion of the total variance in Y explained by the model (ranges from 0 to 1).

## Warnings and Limitations
- **Direction of causation**: Correlation $\neq$ Causation.
- **Multicollinearity**: Explanatory variables are highly correlated with each other.
- **Autocorrelation**: Observations are related over time.
- **Heteroscedasticity**: Error variance is not constant.
- **Misspecification**: Omitted variables or wrong functional form.
