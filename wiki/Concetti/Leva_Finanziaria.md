# ⚡ Financial Leverage

**Financial Leverage** measures the degree to which a firm uses debt to finance its operations. It amplifies both returns and losses.

## Leverage Effect

```
ROE = ROA + (ROA - i) × D/E
```

| Scenario | If ROA > i (cost of debt) | If ROA < i |
|---|---|---|
| **Effect** | Leverage **amplifies** ROE | Leverage **reduces** ROE |
| **Risk** | Superior return | Amplified losses |

## Degrees of Leverage

### Degree of Financial Leverage (DFL)
```
DFL = EBIT / (EBIT - Interest)
```
Measures the sensitivity of EPS to changes in EBIT.

### Degree of Operating Leverage (DOL)
```
DOL = (Revenue - Variable Costs) / EBIT
```
Measures the impact of fixed operating costs.

### Degree of Total Leverage (DTL)
```
DTL = DOL × DFL
```
Combined effect of operating and financial leverage.

## Leverage Ratios

| Ratio | Formula | Interpretation |
|---|---|---|
| **D/E** | Debt / Equity | Higher = more levered |
| **D/A** | Debt / Total Assets | Debt share of assets |
| **Equity Multiplier** | Assets / Equity | = 1 + D/E |
| **ICR** | EBIT / Interest | Debt service capacity |

## Risks of Excessive Leverage

- **Financial Distress**: inability to service debt
- **Bankruptcy Risk**: insolvency proceedings
- **Agency Costs**: conflicts between shareholders and creditors
- **Rating Downgrade**: increase in cost of debt
