## Formule Chiave Estratte (indice rapido)
- **EVA** = (After-tax Interest + Net Income) − (WACC × Total Capital)
- **ROC** = (After-tax Interest + Net Income) / Total Capital
- **ROA** = (After-tax Interest + Net Income) / Total Assets
- **ROE** = Net Income / Equity at start of year
- **MVA** = Market Cap − Book Equity
- **TIE** = EBIT / Interest
- **Cash Coverage** = (EBIT + Depreciation) / Interest
- **Current Ratio** = Current Assets / Current Liabilities
- **Quick Ratio** = (Cash + Securities + Receivables) / Current Liabilities
- **NPV** = C0 + C1/(1+r) + C2/(1+r)^2 + ...
- **IRR**: tasso che rende NPV = 0
- **PI** = NPV / Investment
