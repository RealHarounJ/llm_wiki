# 📚 Source — How is LLM Changing Search on the Web?

Exploration of how LLM-based search transforms the web search paradigm.

## Key Metadata

- **Published**: 2025-08-25
- **Topic**: [[Semantic_Search]], [[RAG]]

## Summary
- Shift from keyword matching to intelligent, context-aware conversations.
- Three types of LLM search systems: Closed-book, Open-book, and [[RAG]] (Hybrid).
- **SEO Evolution**: Shift from ranking on SERPs to being included in AI summaries.
- **RAG**: Retrieval Augmented Generation enables precise, context-based answers using live data.
