# Real Options (Opzioni Reali)

Le **opzioni reali** conferiscono a un'azienda la flessibilità di modificare, adattare o prendere decisioni su un progetto di investimento in corso d'opera, al fine di migliorarne il Valore Attuale Netto (NPV) o mitigare potenziali perdite.

La flessibilità assume un valore maggiore quando l'incertezza futura è elevata.

## 1. Categorie Principali

| Opzione | Descrizione | Esempio |
| :--- | :--- | :--- |
| **Espansione (Expand)** | Permette di aumentare la produzione se la domanda è superiore alle previsioni. | Costruire una fabbrica con spazio extra o opzioni su materie prime. |
| **Abbandono (Abandon)** | Permette di terminare un progetto se le condizioni peggiorano, limitando le perdite. | Progetti con beni tangibili (aerei, immobili) facilmente rivendibili. |
| **Produzione (Production)** | Flessibilità del sistema produttivo per variare quantità, qualità o tipologia di output. | Impianto capace di produrre diverse varianti di un prodotto. |
| **Tempistica (Timing)** | Flessibilità di decidere *quando* avviare l'investimento (subito o posticipare). | Aspettare per raccogliere più informazioni, rischiando però l'azione dei competitor. |

## 2. Strumenti di Analisi: Alberi Decisionali

Per mappare queste opzioni e l'effetto delle decisioni future sui flussi di cassa, si utilizzano gli **alberi decisionali (decision trees)**.
*   Ogni "nodo" rappresenta un punto di decisione strategica.
*   Permettono di visualizzare scenari diversi in risposta agli eventi di mercato.
