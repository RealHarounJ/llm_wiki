# ⏳ Time Value of Money

The **Time Value of Money (TVM)** is the foundational principle of finance: a dollar available today is worth more than a dollar available in the future, because it can be invested to generate returns.

## Core Formula

```
FV = PV × (1 + r)^n
PV = FV / (1 + r)^n
```

Where:
- **PV** = Present Value
- **FV** = Future Value
- **r** = interest/discount rate per period
- **n** = number of periods

## Derived Concepts

- **Discounting**: bringing a future cash flow back to the present
- **Compounding**: projecting a present value into the future
- **Annuity**: series of equal payments at regular intervals
- **Perpetuity**: annuity with infinite duration → `PV = C / r`
