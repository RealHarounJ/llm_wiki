# 💰 Topic 3d — Cash Budgeting (Budget di Cassa)

## 🎯 Cos'è il Cash Budget?
Il budget di cassa è uno **strumento di pianificazione finanziaria a breve termine** che prevede i flussi di cassa (entrate e uscite) su base mensile o trimestrale.

> Obiettivo: Identificare **quando** e **quanti** fondi esterni saranno necessari — o quando ci sarà un surplus da investire.

## 🏗️ Struttura del Cash Budget

```
╔════════════════════════════════════════════════╗
║           CASH BUDGET (Trimestrale)            ║
╠════════════════════════════════════════════════╣
║ ENTRATE                    Q1    Q2    Q3    Q4 ║
║ Incassi da vendite          X     X     X     X ║
║ (+) Altre entrate           X     X     X     X ║
║ = TOTALE ENTRATE            X     X     X     X ║
╠════════════════════════════════════════════════╣
║ USCITE                                         ║
║ Pagamenti a fornitori       X     X     X     X ║
║ Salari e stipendi           X     X     X     X ║
║ Affitti / Utenze            X     X     X     X ║
║ Investimenti capex          X     X     X     X ║
║ Imposte e tasse             X     X     X     X ║
║ = TOTALE USCITE             X     X     X     X ║
╠════════════════════════════════════════════════╣
║ FLUSSO NETTO DI CASSA       X     X     X     X ║
║ (+) Saldo iniziale          X     X     X     X ║
║ = SALDO CUMULATO            X     X     X     X ║
║ (-) Saldo minimo richiesto  X     X     X     X ║
║ = EFN / SURPLUS             X     X     X     X ║
╚════════════════════════════════════════════════╝
```

## 📐 Concetti Chiave

### Timing degli Incassi
Le vendite **non si incassano immediatamente**. Se vendi a credito (es. 30 giorni):
- Vendite di gennaio → incassate a febbraio
- Bisogna stimare la % di vendite incassate nel mese stesso vs nei mesi successivi

### Saldo Minimo di Cassa (Cash Reservoir)
Le aziende mantengono una **riserva di sicurezza** (minimum cash balance) che funge da "paracadute finanziario". Se il saldo previsto scende sotto questa soglia, occorre finanziarsi esternamente.
- Include: Cassa contante, titoli negoziabili (marketable securities) e linee di credito non utilizzate.

### Strategie di Finanziamento a Breve
Se il cash budget mostra un deficit:
1. **Bank Loan**: Utilizzo di linee di credito (revolving).
2. **Stretching Payables**: Ritardare i pagamenti ai fornitori.
    - *Vantaggio*: Liquidità immediata.
    - *Svantaggio*: Costo elevato (perdita di sconti per pagamento immediato, es. 2/10 net 30) e danno reputazionale.
3. **Vendita di Titoli Negoziabili**: Smobilizzo del cash reservoir.
