# 🧠 GPT (Generative Pre-trained Transformer)

A **Generative Pre-trained Transformer** (GPT) is a type of [[Large_Language_Models]] (LLM) that utilizes the [[Transformer_Architecture]] for generative AI tasks.

## The GPT Series (OpenAI)
| Model | Release | Parameters | Notable Features |
|---|---|---|---|
| **GPT-1** | 2018 | 117M | Introduced generative pre-training. |
| **GPT-2** | 2019 | 1.5B | Coherent text generation; "too dangerous to release" initially. |
| **GPT-3** | 2020 | 175B | Few-shot and zero-shot learning abilities. |
| **GPT-4** | 2023 | Trillions (est.) | Multimodal capabilities (text/images). |
| **GPT-5** | 2025 | Unknown | Automatic router for reasoning vs. fast models. |

## Core Principles
1. **Generative Pre-training (GP)**: Self-supervised learning on large, unlabeled datasets.
2. **Fine-tuning**: Adapting the model to specific tasks using labeled data.
3. **RLHF**: Reinforcement Learning from Human Feedback, used to align behavior (crucial for ChatGPT).

## Branding and Legal Status
OpenAI asserts that "GPT" is a brand. However, the USPTO has frequently declined trademark registration, citing that the term is "descriptive and generic" within the AI field.

## Connections
- Direct implementation of [[Transformer_Architecture]].
- Core engine for ChatGPT and various [[Foundation_Models]].
- Competitors include [[Gemini]], Llama, and Claude.
